1. Résumé Exécutif

EuroSalary est une plateforme SaaS de comparaison salariale pan-européenne qui connecte employeurs, recruteurs, professionnels et partenaires API.

Modèle de revenu: Abonnements Premium (B2C), API (B2B), Rapports pays, Solutions entreprise

Cible 2026: €950k revenue annuel | 2k+ utilisateurs premium | 50+ clients B2B
